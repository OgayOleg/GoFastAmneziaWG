GoFastAmneziaWG

Описание проекта
GoFastAmneziaWG — backend-проект на Go с микросервисной архитектурой, предназначенный для управления VPN-туннелями и взаимодействия с пользователями через Telegram.
Проект состоит из двух независимых микросервисов:
AWG-service — сервис управления туннелями (добавление и удаление peer’ов), работающий с повышенными системными привилегиями.
Telegram-service — Telegram-бот для взаимодействия с пользователями и управления бизнес-логикой.
Проект демонстрирует практические навыки backend-разработки, работы с сетью и системными ресурсами, а также принципы изоляции сервисов.

Стек технологий:
  Golang 1.22
  net/http — REST API
  Telegram Bot API
  PostgreSQL (планируется интеграция)
  Makefile
  Linux
  Git
